## Design Context

### Users
- Primary: Students (teens/young adults)
- Context: Seeking mental health support, often dealing with academic stress, anxiety
- Needs: Privacy (anonymous), accessibility (24/7), affordability (free for students)

### Brand Personality
- **Warm & Welcoming**: Friendly, approachable, comforting - like a supportive friend
- **Calming**: Soft, peaceful digital sanctuary
- Trustworthy and non-judgmental

### Aesthetic Direction
- **Soft & Calming**: Pastel colors, gentle design
- Keep existing brand colors (purple/lavender tones from CSS variables)
- Mobile-first with fluid/adaptive sizing using `clamp()` for all screen sizes
- Clean, modern with gentle animations

### Design Principles
1. **Accessibility First**: Large touch targets (44x44px min), readable text sizes, clear visual hierarchy
2. **Privacy-Conscious**: Anonymous features prominent, no judgment
3. **Mobile-Optimized**: Thumb-friendly navigation, bottom-anchored controls, responsive layouts
4. **Calm Aesthetic**: Generous whitespace, soft colors, smooth transitions
5. **Fluid Scaling**: Use `clamp()` for typography and spacing to adapt across all devices
